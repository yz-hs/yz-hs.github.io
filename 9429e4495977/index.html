链式前向星

作者：yz-hs。 本文章共有 670 字。 本文写于 2019 年 08 月 30 日 。

1.简介

——$linked-forward-star$

是邻接表的静态实现。

它的优点是节省了分配内存的时间，效率更高。

2.实现

链式前向星的构成由一个结构体（包括目标点、边权值和下一个同起点的边）和head数组（用于存放某点的第一条出边）

必要的时候还可以添加一个统计入度的数组，因为进行BFS DFS的时候是依靠点的出度和出边的邻接关系来进行的。假如有多于一个点的入度为0，那么将只能遍历到其中一个点以及往后的内容。

对于链式前向星，总的一句话：链式前向星每添加一条边就会更新head数组，使得head数组存放的总是最新添加的某点的出边，此出边的next总指向head数组之前存放的出边的序号。

具体：

nxt: 以本点为前驱点的下一条边的数组地址
len: 边权
end: 后继点
cnt: 计数

3.膜板

建立

class edge
{
      public:
           int nxt,len,end;
}e[5000000];//画猴子
int head[205]={0},cnt=0;//打井
void add(int x,int y,int len)
{
     e[++cnt]=(edge){head[x],len,y};
     head[x]=cnt;
}

其中add是指连一条从x到y边权为len的边

遍历

for(int i=head[...];i;i=e[i].nxt)//提猴子
{
    //do something...
}
