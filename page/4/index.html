快速幂

以下以求 aaa 的 111111 次方来介绍。把 111111 转换成二进制数 1011(2)1011_{(2)}1011(2)​。设其第 iii 位的权为 $ 2^{i-1} $。因此，我们将 a11a^{11}a11 转化为算 a20+a21+a23a^{2^0}+a^{2^1}+a^{2^3}a20+a21+a23。

ll qpow(ll a,ll p,ll m)
{
    ll r=1;
    while(p)
    {
        r=(p%2?r*a:r)%m;
        p/=2; a=a*a%m;
    }
    return r%m;
}
