第一问

注意到把第 nnn 行的数字拆成两个因子相乘的形式之后，其和是定值 n+3n+3n+3。这个性质也是解决全部四问的关键，因为这是本题的 idea 核心所在。

Sx=(∑i=1ni(n−i))−2(n−1)=n(∑i=1ni)−(∑i=1ni2)−2(n−1)=12n2(n+1)−16n(n+1)(2n+1)−2(n−1)=16(n3−13n+12)\large\begin{aligned} S_x&=\left(\sum_{i=1}^ni(n-i)\right)-2(n-1)\\ &=n\left(\sum_{i=1}^ni\right)-\left(\sum_{i=1}^ni^2\right)-2(n-1)\\ &=\frac{1}{2}n^2(n+1)-\frac{1}{6}n(n+1)(2n+1)-2(n-1)\\ &=\frac{1}{6}(n^3-13n+12) \end{aligned}Sx​​=⎝⎛​i=1∑n​i(n−i)⎠⎞​−2(n−1)=n⎝⎛​i=1∑n​i⎠⎞​−⎝⎛​i=1∑n​i2⎠⎞​−2(n−1)=21​n2(n+1)−61​n(n+1)(2n+1)−2(n−1)=61​(n3−13n+12)​

然后我们就做完了？

但是注意到数字很大，一乘就会爆掉，无法应用第四个式子。并且模数 ppp 不一定是质数，没法求逆元。用高精度呢，不仅麻烦，而且有 TLE 风险。我们只能对数进行手动除，配合龟速乘解决这一问。

如果直接应用第三个式子，那么有：

  • n,n+1,2n+1n,n+1,2n+1n,n+1,2n+1 中必定有一个可以被 333 整除的数。

那么，我们就可以在不爆掉的情况准确计算了。但是，这样常数大，容易 TLE。

第二问

由于数字三角形的对称性，求 ai+1,j+1a_{i+1,j+1}ai+1,j+1​ 的最值相当于求 ai+1,ja_{i+1,j}ai+1,j​ 的最值。

数字三角形的第 iii 列是公差为 i+1i+1i+1 的等差数列。那么，首先得出的一个结论是：一个数不存在于数字三角形等价于它是质数。然后，你会发现 ai,ja_{i,j}ai,j​ 与 ai+1,ja_{i+1,j}ai+1,j​ 在同一列上。那就意味着，它们两个数在一个等差数列上。

显然，最小值是 ai,ja_{i,j}ai,j​ 加最小可能公差。最小可能公差又等于 ai,ja_{i,j}ai,j​ 的最小质因子。所以，只需要使用线性筛筛出每个数的最小质因子就行了。最大值，就是加最大因子，也就是加上 ai,ja_{i,j}ai,j​ 除以最小质因子。

现在说明一下如何用线性筛求最小质因子。

考虑线性筛的时间复杂度是线性的特点。为什么时间复杂度是线性的的呢？原因是，合数被且仅被它的最小质因子筛掉。所以一个数被筛掉时，顺便就可以记录下来它的最小质因子。

另外，一个事实是，线性筛中，当满足 i%prime[j]==0 时，就要 break。其原因就是此时 i*prime[j] 的最小质因子就是 prime[j]。如果继续枚举质数，那么接下来的数 i*prime[j+1] 的最小质因子就不是 prime[j+1] 了，那么线性复杂度就无法保证。

	{
		ll ret=0;

		a%=p; b%=p;

			ret+=a*(b%10)%p;
			b/=10; a=a*10%p;
		}

I 掷骰子

题目背景

https://www.luogu.com.cn/problem/U239990?contestId=81097

巨佬 zzy 是神，祂所处的世界是常人无法理解的高维空间。一天，巨佬 zzy 穿越到了我们的世界，并找到蒟蒻 siu，想要和她进行掷骰子游戏。

当然，zzy 的骰子也是常人无法理解的——由于它是高维物体，它可以等可能地掷出从 111 到 nnn 的任意点数！

蒟蒻 siu 被 zzy 非凡的智慧所震撼，所以她想要问你一个问题。因为 zzy 急着要吊打 siu，所以 zzy 只给你 1s\tt1s1s 的时间回答她的问题。

题目描述

巨佬 zzy 的骰子可以等可能地掷出从 111 到 nnn 的任意点数。

掷骰子游戏是这样的：siu 连续掷 aaa 次骰子，取掷出的最小点数为 siu 的得分。为了游戏的趣味性和易操作性，限制 aaa 要么是 222，要么是 333。

蒟蒻 siu 想知道自己的期望得分是多少。为了避免回答小数（避免有理数取模），你需要将答案乘上 nan^ana。为了避免答案太大，你只需回答答案对 ppp 取模的结果即可。

由于“期望”这个概念蒟蒻 siu 连听过都没听过，所以她准备了一份说人话的简明题意。

简明题意：

给定 nnn，aaa，ppp。当 a=2a=2a=2 时，求：

(∑i=1n∑j=1nmin⁡(i,j)) mod p\large\left(\sum_{i=1}^n\sum_{j=1}^n \min(i,j)\right) \bmod p⎝⎛​i=1∑n​j=1∑n​min(i,j)⎠⎞​modp

当 a=3a=3a=3 时，求：

(∑i=1n∑j=1n∑k=1nmin⁡(i,j,k)) mod p\large\left(\sum_{i=1}^n\sum_{j=1}^n\sum_{k=1}^n \min(i,j,k)\right) \bmod p⎝⎛​i=1∑n​j=1∑n​k=1∑n​min(i,j,k)⎠⎞​modp
