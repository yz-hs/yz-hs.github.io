计算几何简介

需要计算的几何。即讨论点、线、面之间的相互关系，比如线段之间的相交关系，多边形的面积等。

做计算几何题是一项艰苦的体力劳动。计算几何就是让你想到但写不出来。

“说的比做的容易得多得多……”

向量

向量，同矢量，⼀个有方向、大小的量。数对应着标量。

二维平面上，⼀个点 AAA 与向量 OA⃗=a⃗=(xA,yA)\vec{OA} = \vec{a} = (x_A, y_A)OA=a=(xA​,yA​) 等价，程序实现时不对点与向量作区分（向量的坐标表示）。向量通常用小写字母表示。

设 A(xA,yA)A(x_A, y_A)A(xA​,yA​)，B(xB,yB)B(x_B, y_B)B(xB​,yB​)，则向量 AB⃗=B−A=(xB−xA,yB−yA)\vec{AB} = B-A = (x_B-x_A, y_B-y_A)AB=B−A=(xB​−xA​,yB​−yA​)。

一些内容

  1. 模长 ∣a⃗∣|\vec{a}|∣a∣：a⃗\vec{a}a的长度。

  2. 零向量 0⃗\vec{0}0：∣0⃗∣=0|\vec{0}|=0∣0∣=0。

  3. 单位向量：∣a⃗∣=1|\vec{a}|=1∣a∣=1。

  4. 共线（平行）向量：方向相同或相反。

  5. 相等向量：方向相同，长度相同。

  6. 旋转：向量 (x,y)(x, y)(x,y) 绕坐标原点逆时针旋转 θθθ 度。

  7. 平面向量基本定理、基底、夹角、垂直、正交分解、投影……

运算

线性运算

加，减，数乘。

点积

a⃗⋅b⃗=∣a⃗∣∣b⃗∣cos⁡θ\vec{a}\cdot \vec{b} = |\vec{a}| |\vec{b}| \cos θa⋅b=∣a∣∣b∣cosθ

其中 θθθ 表示由 a⃗\vec{a}a旋转到 b⃗\vec{b}b所经过的夹角。其绝对值等于 a⃗\vec{a}a在 b⃗\vec{b}b上的投影的模长乘上向量 b⃗\vec{b}b的模长。

设 a⃗=(x1,y1)\vec{a} = (x_1, y_1)a=(x1​,y1​)，b⃗=(x2,y2)\vec{b} = (x_2, y_2)b=(x2​,y2​)，则它们的点积等于：

x1x2+y1y2x_1x_2 + y_1y_2x1​x2​+y1​y2​

简单应用：两个向量垂直 ⇔⇔⇔ 其点积为 000。

叉积

a⃗⋅b⃗=∣a⃗∣∣b⃗∣sin⁡θ\vec{a}\cdot \vec{b} = |\vec{a}| |\vec{b}| \sin θa⋅b=∣a∣∣b∣sinθ

几何意义为，以两个向量为邻边构成的平行四边形的有向面积。当 b⃗\vec{b}b在 a⃗\vec{a}a左侧时为正，右侧时为负（其方向遵循右手定则）。

设 a⃗=(x1,y1)\vec{a} = (x_1, y_1)a=(x1​,y1​)，b⃗=(x2,y2)\vec{b} = (x_2, y_2)b=(x2​,y2​)，则它们的点积等于：

x1y2−x2y1x_1y_2 - x_2y_1x1​y2​−x2​y1​

简单应用：两个向量平行 ⇔⇔⇔ 其叉积为 0⃗\vec{0}0。

向量的旋转

把向量 a⃗=(x,y)\vec{a} = (x, y)a=(x,y) 旋转 θθθ 弧度后得到的坐标为：

(xcos⁡θ−ysin⁡θ,xsin⁡θ+ycos⁡θ)(x \cos θ - y \sin θ, x \sin θ + y \cos θ)(xcosθ−ysinθ,xsinθ+ycosθ)
