“丹钓战”，顾名思义，就是单调栈。所以本题用到了单调栈。

题意

  • 有 nnn 个二元组 (ai,bi)(a_i, b_i)(ai​,bi​)，编号为 111 到 nnn。
  • 有一个初始为空的栈 SSS，向其中加入元素 (ai,bi)(a_i, b_i)(ai​,bi​) 时，先不断弹出栈顶元素直至栈空或栈顶元素 (aj,bj)(a_j , b_j)(aj​,bj​) 满足 ai≠aja_i \neq a_jai​=aj​ 且 bi<bjb_i < b_jbi​<bj​，然后再将其加入栈中。
  • 如果一个二元组入栈后栈内只有这一个元素，则称该二元组是“成功的”。
  • 有 qqq 个询问 [li,ri][l_i, r_i][li​,ri​]，询问若将编号在 [li,ri][l_i, r_i][li​,ri​] 中的二元组按编号从小到大依次入栈，“成功的”二元组数目。询问之间相互独立。
  • 对于所有测试点：1≤n,q≤5×1051 \leq n, q \leq 5 \times 10^51≤n,q≤5×105，1≤ai,bi≤n1 \leq a_i, b_i \leq n1≤ai​,bi​≤n，1≤li≤ri≤n1 \leq l_i \leq r_i \leq n1≤li​≤ri​≤n。

题解 - 简略版

提供一个在线的时间复杂度为 O(nlog⁡n+q)\mathcal{O}(n\log n+q)O(nlogn+q) 的算法。

下面提到的“栈”指题面中的“栈”。令所有元素依次入栈，设当第 iii 号二元组入栈时，栈内元素数量为 aia_iai​。按编号顺序，所有的 aia_iai​ 形成了一个数列 AAA。

那么对于任意一个询问区间 [l,r][l,r][l,r]，假如第 xxx 号二元组是“成功的”，那么 axa_xax​ 满足：

ax≤min⁡i=ljx(ai)\large a_x\le\min_{i=l_j}^{x}(a_i)ax​≤i=lj​minx​(ai​)

我们称这样的 axa_xax​ 是“成功的”，那么，对于这个区间，所有“成功的” axa_xax​ 按编号顺序将会排列成一个单调不升序列 Sl,rS_{l,r}Sl,r​，这个区间的答案就是 ∣Sl,r∣|S_{l,r}|∣Sl,r​∣。

在 SSS 中，第 iii 个数 bib_{i}bi​ 和数 bi+1b_{i+1}bi+1​ 必定满足：假设 bib_ibi​ 对应 AAA 中的 apa_pap​，bi+1b_{i+1}bi+1​ 对应 aqa_qaq​，那么 apa_pap​ 后面第一个小于等于 apa_pap​ 的数是 aqa_qaq​。我们称 apa_pap​ 的 next 是 aqa_qaq​。所有 aia_iai​ 的 next 可以通过单调栈求得。

注：这个求 next 的模板题，可以在 luogu 上搜 单调栈。next 这个名字是我自己起的。

设 f(i,j)=∣Si,j∣f(i,j)=|S_{i,j}|f(i,j)=∣Si,j​∣。如果所有 aia_iai​ 向它的 next 连边，这样会构造出一个森林。以所有没有 next 的 axa_xax​ 为根节点（设其深度为 111），那么 aia_iai​ 的深度就是 f(i,n)f(i,n)f(i,n) 的值。

如果 aj∈Si,na_j\in S_{i,n}aj​∈Si,n​（j≥ij\ge ij≥i），那么必有：

f(i,j)=f(i,n)−f(j,n)+1f(i,j)=f(i,n)-f(j,n)+1f(i,j)=f(i,n)−f(j,n)+1

设 aw∈Si,ja_w\in S_{i,j}aw​∈Si,j​ 是 Si,jS_{i,j}Si,j​ 中的最小值（若有多个最小值，规定最靠近 jjj 的为最小值），那么必有：

f(i,j)=f(i,w)=f(i,n)−f(w,n)+1f(i,j)=f(i,w)=f(i,n)-f(w,n)+1f(i,j)=f(i,w)=f(i,n)−f(w,n)+1

用 ST 表维护区间最小值即可。

详细版

题解 - 详细版

提供一个时间复杂度为 O(nlog⁡n+q)\mathcal{O}(n\log n+q)O(nlogn+q) 的算法，大致是把问题转化为静态 RMQ 问题，利用 ST 表解决。其瓶颈也在于此。

拿到这个题，我先打暴力，也就是按照题意模拟。

之后，我发现如果令所有元素依次入栈，则所询问的区间 [l,r][l,r][l,r] 内的二元组入栈显然是依次且连续的；在区间 [l,r][l,r][l,r] 内元素依次入栈时，不会有区间后面（即编号大于 rrr）的二元组压入，可能会有在区间前面（即编号小于 lll）的一些二元组弹出。

下面所提的“栈”，均指从 111 到 nnn 进行整个过程的栈。

所以我想，接下来应当研究如何根据每个过程栈的某些信息，来降低得到区间 [l,r][l,r][l,r] 的答案的时间复杂度。

如何判定二元组是否是“成功的”呢？

首先看“成功”的定义：如果一个二元组入栈后栈内只有这一个元素，则称该二元组是“成功的”。

对于一次询问 [l,r][l,r][l,r]，显然，第 lll 号二元组一定是“成功的”。如果我们说栈的大小指栈中元素的数量，那么按照“成功”的定义，如果接下来的第 l+1l+1l+1 号二元组压入后，栈的大小不大于压入 lll 时栈的大小（因为可能会有一些二元组被弹出），那么它也是“成功的”。

当第 iii 个二元组压入后，如果此时栈的大小不大于 [l,i−1][l,i-1][l,i−1] 区间上各元素压入时栈的大小的最小值（这样才能保证符合题意，即当时确实是只有 iii 这一个二元组），那么它才是“成功的”。显然这些“成功的”二元组对应的栈的大小形成了一条单调不升序列，这个序列的长度即为答案。

通过对判定二元组“成功”的研究，我们可以发现不在 [l,r][l,r][l,r] 内的二元组，对于判断 [l,r][l,r][l,r] 中二元组的“成功”与否，是没有影响的。

那么我们可以转化问题了。

设 aia_iai​ 表示压入第 iii 个二元组时，栈的大小（显然可以 O(n)\mathcal{O}(n)O(n) 得到所有的 aia_iai​）。那么问题转化为：

给定数列 aaa，及 qqq 组询问，每次询问给定的 [l,r][l,r][l,r] 区间上以 lll 开头的一条单调不升序列的长度。

为了方便，我把这个特殊的单调不升序列称为（[l,r][l,r][l,r] 区间上的）S 序列。这是个怎样的序列呢？这个 S 序列有这样的性质——“可减性”。

设 f(i,j)f(i,j)f(i,j) 表示 [i,j][i,j][i,j]（j≥ij\ge ij≥i）区间上的 S 序列的长度；特殊地， f(i)f(i)f(i) 表示 [i,n][i,n][i,n] 区间上 S 序列的长度。如果 aja_jaj​ 在 [i,n][i,n][i,n] 的 S 序列上，那么必有：

f(i)−f(j)+1=f(i,j)f(i)-f(j)+1=f(i,j)f(i)−f(j)+1=f(i,j)

这个结论是显然的。由于二元组入栈依次且连续，这就决定 S 序列上连续的两个数 p,qp,qp,q（p≥qp\ge qp≥q），在原数列上必定满足：qqq 是 ppp 之后第一个小于等于 ppp 的数。为了方便，把 qqq 称为 ppp 的 next。显然一个数的 next 最多只有一个，所以 [j,n][j,n][j,n] 上的 S 序列必定是 [i,n][i,n][i,n] 上的 S 序列的末尾。因此就有了这个结论。

那么如何求得 f(i)f(i)f(i) 呢？

我们可以通过丹钓战单调栈以 O(n)\mathcal{O}(n)O(n) 来找到所有 aia_iai​ 的 next。单调栈维护自栈底到栈顶单调递增的序列，其中的元素是数字。如果待压入元素小于等于栈顶，则显然此时栈顶的 next 就是待压入元素，弹出栈顶，直至待压入元素大于栈顶或栈为空。正确性显然。

如果所有 aia_iai​ 的 next 向 aia_iai​ 连边，显然就形成了一个树形结构，即森林（每个节点出度可大于 111，入度最大为 111）。以入度为 000 的 aia_iai​ 为根节点，iii 的深度即为 f(i)f(i)f(i) 的值。

现在关键是如何得到 f(l,r)f(l,r)f(l,r) 也就是答案。

这里引入一个显然的结论：设 xxx 是区间 [l,r][l,r][l,r] 上的最小值，则 xxx 一定包含在 S 序列中。假设 xxx 不在 S 序列中，且不是第 lll 个数（如果是第 lll 个数，那么显然它是 S 序列的开头，因为第 lll 号二元组一定是“成功的”），则 xxx 前面必有一个数 yyy 在 S 序列里面。既然 xxx 和 yyy 之间的数没有在 S 序列里面，则必定它们之间的数比 yyy 大。那么显然 xxx 就成为了 yyy 的 next，和假设矛盾。

显然，区间中，值为 xxx 的数必定是这个区间中 S 序列中的最后一个数（这里的 xxx 同上段文字）。设这个区间最后一个值为 xxx 的数的位置是 www，则区间 [l,r][l,r][l,r] 的答案显然是：

f(l,r)=f(l,w)=f(l)−f(w)+1f(l,r)=f(l,w)=f(l)-f(w)+1f(l,r)=f(l,w)=f(l)−f(w)+1

如何得知这个 www 呢？

显然，这已经是一个静态 RMQ 问题了。

定义二元组 (ai,i)(a_i,i)(ai​,i) 的大小关系为：(ai,i)<(aj,j)(a_i,i)<(a_j,j)(ai​,i)<(aj​,j) 当且仅当 ai<aja_i<a_jai​<aj​，或 ai=aja_i=a_jai​=aj​ 时 i>ji>ji>j。使用能维护区间最值的数据结构（静态即可，例如 ST 表）维护 [1,n][1,n][1,n] 的二元组即可。这样定义大小关系，就能保证维护的确实是一个区间的 www。

时间复杂度：O(nlog⁡n+q)\mathcal{O}(n\log n+q)O(nlogn+q)。O(nlog⁡n)\mathcal{O}(n\log n)O(nlogn) 的预处理，O(1)\mathcal{O}(1)O(1) 的查询。

使用线段树，时间复杂度则为：O(n+qlog⁡n)\mathcal{O}(n+q\log n)O(n+qlogn)。O(n)\mathcal{O}(n)O(n) 的预处理，O(log⁡n)\mathcal{O}(\log n)O(logn) 的查询。

代码

  • num[]：AAA 数列。
  • f[]：f(i,n)f(i,n)f(i,n)。
#include<bits/stdc++.h>
using namespace std;
namespace _
{
    typedef int ll;
    const ll N=500000+10,inf=2147483640,V=19;
    struct P{ll a,b;}p[N],stk[N],st[V+5][N];
    struct edge{ll nxt,to;}e[N];
    ll n,q,head[N],cnt=0,num[N],f[N],top=0,b[N];
    inline ll read()
    {
        ll s=0,w=1;
        char ch=getchar();
        while(ch<'0'||ch>'9')
        {
            if(ch=='-')
                w=-1;
            ch=getchar();
        }
        while(ch>='0'&&ch<='9')
            s=s*10+ch-'0',ch=getchar();
        return s*w;
    }
    inline void write(ll x)
    {
        if(x>9) 
            write(x/10);
        putchar(x%10+'0');
    }
    P min(P a,P b)//规定大小
    {
        if(a.a<b.a) return a;
        else if(a.a==b.a)
        {
            if(a.b<b.b) return b;
            return a;
        }
        return b;
    }
    void add(ll a,ll b)
    {
        e[++cnt]=(edge){head[a],b};
        head[a]=cnt; 
    }
    void main()
    {
        ll x,y,len;
        n=read(); q=read();
        for(ll i=1;i<=n;i++) p[i].a=read();
        for(ll i=1;i<=n;i++) p[i].b=read();
        for(ll i=1;i<=n;i++)
        {
            while(top>=1&&!(stk[top].a!=p[i].a&&stk[top].b>p[i].b)) top--;
            stk[++top]=p[i]; num[i]=top;//求 A 数列
        }
        
        //构造 ST 表：
        for(int i=1;i<=n;i++)
            st[0][i]=(P){num[i],i};
        for(int i=1;i<=V;i++)
            for(int j=1;j<=n;j++)
                st[i][j]=min(st[i-1][j],st[i-1][j+(1<<(i-1))]);
        for(int i=1;i<=V;i++)
            b[1<<i]=1;
        for(int i=1;i<=n;i++)
            b[i]+=b[i-1];
        
        top=0;
        for(ll i=1;i<=n;i++)
        {
            while(top>=1&&stk[top].a>=num[i])
                add(i,stk[top].b),top--;
            //求每个 a_i 的 next，连边
            stk[++top]=(P){num[i],i};
        }
        
        //求 f(i,n)
        for(ll i=n;i>=1;i--)
        {
            if(f[i]==0) f[i]=1;
            for(ll j=head[i];j;j=e[j].nxt)
            	f[e[j].to]=f[i]+1;
        }
        
        for(ll i=1;i<=q;i++)
        {
            x=read(); y=read(); len=y-x+1;
            write(f[x]-f[min(st[b[len]][x],st[b[len]][y+1-(1<<b[len])]).b]+1);
            //f(x,y)=f(x,w)=f(x,n)-f(w,n)+1
            putchar('\n');
        }
        return;
    }
}
int main()
{
    _::main();
    return 0;
}
