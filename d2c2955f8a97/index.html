P2347 砝码称重

题目描述

设有 1g1g1g、2g2g2g、3g3g3g、5g5g5g、10g10g10g、20g20g20g 的砝码各若干枚（其总重 ≤1000\le1000≤1000 ），

输入格式

输入方式：a1a_1a1​ , a2a_2a2​ , a3a_3a3​ , a4a_4a4​ , a5a_5a5​ , a6a_6a6​

（表示 1g1g1g 砝码有 a1a_1a1​ 个，2g2g2g 砝码有 a2a_2a2​ 个，…，20g20g20g 砝码有 a6a_6a6​ 个）

输出格式

输出方式：Total=NTotal=NTotal=N

（NNN 表示用这些砝码能称出的不同重量的个数，但不包括一个砝码也不用的情况）

设有 1g1g1g、2g2g2g、3g3g3g、5g5g5g、10g10g10g、20g20g20g 的砝码各若干枚

即表示 1g1g1g 砝码有 a1a_1a1​ 个，2g2g2g 砝码有 a2a_2a2​ 个，…，20g20g20g 砝码有 a6a_6a6​ 个

emm…这不就是多重背包吗？

我们可以设有 maxmaxmax 个背包，整齐地排成一行，分别可以承重 1,2,3,...,max⁡1,2,3,...,\max1,2,3,...,max（其中 max⁡\maxmax 指所有砝码的重量）。

跑一遍多重背包，数一下各个背包，有几个背包放满了砝码（bag[i]==i）！

补充更新：这其实是可行性 dp。状态转移方程的逻辑是：既然 bag[i] 是合法的，那么在 iii 的基础上再加一个砝码也是合法的。

#include<bits/stdc++.h>
#define _ 0
using namespace std;
int w[]={0,1,2,3,5,10,20},a[10],f[1005];
int main()
{
	int ans=0;
	for(int i=1;i<=6;i++)
		scanf("%d",&a[i]);
    for(int i=1;i<=6;i++)//dp,多重背包
	 	for(int j=1000;j>=w[i];j--)
    		for(int k=1;k<=a[i];k++)
    			if(j>=k*w[i])
	   				f[j]=max(f[j],f[j-k*w[i]]+k*w[i]);
	for(int i=1;i<=1000;i++)
		if(f[i]==i)
			ans++;
    printf("Total=%d",ans);
  	return ~~(0^_^0);//嘤嘤嘤
}

P5020 货币系统

在网友的国度中共有 nnn 种不同面额的货币，第 iii 种货币的面额为 aia_iai​，你可以假设每一种货币都有无穷多张。

#include<bits/stdc++.h>

using namespace std;
int t,n,a[100+5],f[25000+5],ans,ins[25000+5];
void work()
{
	for(int i=1;i<=n;i++)
	 	for(int j=a[i];j<=a[n];j++)//小小剪枝，只需枚到货币面额的最大值。
	 	{
	 		f[j]=max(f[j],f[j-a[i]]+a[i]);
	 		if(f[j]==j&&f[j-a[i]]!=0)//f[j]的状态不是由空背包状态转移来，即不是被本面额表示
	 			ins[j]=1;
		}
	return;		
}	
int main()
{ 
	scanf("%d",&t);

	{
		memset(f,0,sizeof(f)),memset(ins,0,sizeof(ins));//记得清0啊啊啊
		scanf("%d",&n),ans=n;

}

证明：$ B \subseteq A$

设有两个货币系统 (n,a)(n,a)(n,a) 和 (m,b)(m,b)(m,b)。

首先，我们需证明：为什么不能添加新的货币面额，而只能减少现有货币面额种类？

即，设 (n,a)(n,a)(n,a) 为集合 AAA，(m,b)(m,b)(m,b) 为集合 BBB，为什么 $ B \subseteq A$？

这里，引用大佬的证明，链接here：

------------------------------以下为引用内容------------------------------

在证明这个猜测前我们先来证明另一个猜测 AAA 集合内不能被其它数组成的数必然存在于 BBB 集合内，这里使用反证法证明。

证明过程

我们设 x∈Ax\in Ax∈A 且 xxx 不能被 AAA 集合内除它以外的元素组成。

然后我们假设 x∉Bx \notin Bx∈/B ，那么就说明BBB集合中必然存在一些元素能够组成 xxx 。

那么这些元素至少存在一个不在集合 AAA 内并且不能被集合 AAA 里的元素组成的数（因为如果不存在的话集合 AAA 内的元素就可以组成 xxx了），可以看到这与集合 BBB 的定义产生了矛盾。

综上所述， AAA 集合内不能被其它数组成的数必然存在于 BBB 集合内 。证毕。

通过这个结论我们便可以证明 B⊆AB \subseteq AB⊆A 这个猜测了。这里同样使用反证法证明。

证明过程

假设存在 x∈Bx \in Bx∈B 且满足 x∉Ax \notin Ax∈/A 。

根据题目条件，xxx 必然可以被 AAA 集合内的数字所组成。

那么必然存在 a1,a2,...aq∈Aa_1,a_2,...a_q \in Aa1​,a2​,...aq​∈A 可以用来组成 xxx 并且这些元素都不能被 AAA 集合内的元素组成（因为如果 aia_iai​ 能被 AAA 集合内其它数组成，那么必然可以用那些数来代替 aia_iai​），根据上一个结论我们可以知道 a1,a2,...aq∈Ba_1,a_2,...a_q \in Ba1​,a2​,...aq​∈B 。

那么也就是说 xxx 可以被 BBB 集合内的数字组成，那么凡是可以用 xxx 组成的数都可以被 BBB 集合内的其它数字组成，那么也就是说即便从集合 BBB 中去掉 xxx 元素也依旧满足条件，这就与 BBB 集合的定义矛盾。

所以对于任意 x∈Bx \in Bx∈B必然满足 x∈Ax \in Ax∈A，即 B⊆AB \subseteq AB⊆A。

那么做这道题仅仅只需要最后一个证明了，如果你到这里都看懂了的话那应该不难想到最后一个猜测就是在 AAA 集合中能被其它数组成的数必然不在 BBB 集合内，事实上这个结论很显然，读者只需要稍微想想便可以想出来，这里就不写证明过程了。
