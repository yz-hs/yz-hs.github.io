Vim 的使用

甲鱼 2021-02-17, 11:14:31 318 隐藏左右两栏 展示左右两栏
首页 / Vim 的使用
分享到

改编自 Vim 入门基础。

配置

在终端输入vim ~/.vimrc 然后添加配置代码：

set ts=4
set sw=4
set smarttab
set number
color desert
syntax on
set cindent
set guifont=Consolas:h16

简介

进入Vim编辑程序，可以在终端输入下面的命令：

$ vim [filename]

其中filename是要编辑器的文件的路径名。如果文件不存在，它将为你建立一个新文件。Vim编辑程序有三种操作模式，分别称为 编辑模式、插入模式 和 命令模式，当运行Vim时，首先进入编辑模式。

速查

编辑模式

k 上移 | j 下移 | h 左移 | l 右移
gg         第一行。
G          最后一行。
dd         删除一行。
ggVG       全选。
NG / Ngg   第N行。
x          删除光标所指向的当前字符。
u          撤消前一条命令的结果。
.          重复最后一条修改正文的命令。
/str       正向搜索字符串 str。
?str       反向搜索字符串 str。
i          在光标左侧插入正文->插入模式。
:          ->命令模式。
"+x        剪切。
"+y        复制。
"+gP       粘贴。

插入模式

ESC / Ctrl+[      退出->编辑模式。

命令模式

:e [DIR]      在已经启动的Vim中打开一个文件。
:wq           保存并退出。
:n            将光标移到第 n 行。
